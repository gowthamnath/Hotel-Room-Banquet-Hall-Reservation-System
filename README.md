Hotel Room and Banquet Hall Reservation System — Project Description
The Hotel Room and Banquet Hall Reservation System is a desktop-based management application developed using Python, CustomTkinter, and MySQL to automate and streamline hotel and event reservation operations. The system provides role-based access for Guests, Staff, and Administrators, allowing each user to perform specific tasks through dedicated dashboards and interfaces.
Guests can create accounts, securely log in, search for available hotel rooms and banquet halls, and make reservations based on their requirements. The system enables users to manage bookings, view reservation details, and access a simple and user-friendly reservation process.
Staff members are responsible for managing operational activities, including approving guest check-ins and check-outs, handling banquet hall reservations, updating booking statuses, and processing customer payments. These features help improve operational efficiency and customer service within the hotel environment.
Administrators have full control over the system and can manage users, assign permissions, monitor reservation activities, and generate reports related to bookings, payments, customers, and overall hotel operations. The admin module supports secure system management and effective decision-making.
The application integrates with a MySQL database to store and manage user accounts, reservation records, room details, banquet hall information, payment transactions, and operational data in real time. The graphical user interface is designed using CustomTkinter to provide a modern, responsive, and interactive desktop experience.

Key Features
User registration and secure login system
Role-based access for Guests, Staff, and Administrators
Hotel room and banquet hall reservation management
Room and hall availability checking
Check-in and check-out approval process
Payment processing and booking status updates
User and permission management
Reservation and payment report generation
MySQL database integration
Multi-screen desktop GUI application using Python and CustomTkinter


This project demonstrates the implementation of database management, GUI development, authentication systems, CRUD operations, and modular software architecture while providing a practical solution for hotel room and banquet hall reservation management.
